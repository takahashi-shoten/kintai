# プロジェクト: 勤怠管理アプリ (kintai)

## 概要

建設業・大工業向けの勤怠記録 PWA。スマホ音声入力で毎日の勤怠を記録・集計する。

## 現状

| 項目 | 内容 |
|---|---|
| ステータス | 稼働中（v1） |
| 技術構成 | HTML/CSS/JS 単一ファイル（index.html）、localStorage |
| 対応端末 | iPhone Safari / Android Chrome / PC ブラウザ |

## 完了済み機能

- [x] 音声入力による勤怠記録
- [x] 月別一覧・集計
- [x] 社長確認画面・CSV出力
- [x] PWA対応（オフライン・ホーム画面追加）
