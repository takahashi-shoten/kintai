# ポートフォリオ：kintai アプリ紹介文

Coconalaのプロフィール・サービスページに掲載する実績紹介。

---

## ポートフォリオ説明文（コピペ用）

```
【制作実績】建設業向け勤怠管理アプリ

■ 依頼内容
建設・大工業の現場従業員が毎日の勤怠をスマホで手軽に記録・集計できるアプリ。
「紙やLINEでのやり取りをなくしたい」「月末の集計作業を自動化したい」という
現場の声から開発が始まりました。

■ 主な機能
・音声入力対応（スマホのマイクで話すだけで記録できる）
・現場名・勤務時間・交通費・遠方経費を一括入力・自動解析
・月別一覧・現場別集計・実働時間の自動計算
・社長確認画面とCSV出力（Excel連携）
・ホーム画面に追加してアプリのように常駐（PWA対応）

■ 対応端末
iPhone Safari / Android Chrome / PC ブラウザ

■ 技術構成
HTML / CSS / JavaScript（シングルファイル構成、インストール不要）

■ こだわったポイント
建設現場では手が汚れていることも多い。だから入力はシンプルに最小限に。
音声入力一発で「8時から17時、交通費3000円、砂川市 現場」と話すだけで
全項目が自動入力される仕様にしました。

■ 納品後の状況
現在も現場で毎日使われています。
月末の集計作業が「ゼロ」になったと好評をいただいています。
```

## スクリーンショット用メモ（実際に撮影して添付する）

Coconalaのポートフォリオには画像添付が効果的。
以下の画面を撮影・スクリーンショットして添付すること：

1. **トップ画面**（日付表示、入力ボタン）
2. **音声入力画面**（テキストが自動入力される様子）
3. **月別一覧画面**（記録が並んでいる状態）
4. **月別集計画面**（合計時間・交通費の集計）
5. **スマホでの表示**（ホーム画面に追加した状態）
