# 問い合わせ対応テンプレート集

Coconalaで問い合わせが来たとき、即座に使える返信文。
アインシュタインが状況に応じてカスタマイズして使う。

---

## テンプレート①：最初の問い合わせへの返信（汎用）

```
◯◯様

はじめまして。ご連絡いただきありがとうございます。

ご状況をもう少し教えていただけますか？

① 業種・業態（どんなお仕事をされていますか？）
② 解決したい課題（何が大変・不便と感じていますか？）
③ 現在の管理方法（紙・Excel・LINEなど）
④ 利用人数（自分だけ / 数名 / 10名以上）
⑤ ご希望の納期・予算のイメージ

上記をざっくりで構いません。いただいた情報をもとに、
最適なご提案と正確なお見積もりをお送りします。

よろしくお願いいたします。
AIC開発室
```

## テンプレート②：勤怠管理の問い合わせ（建設業・工事業）

```
◯◯様

ご連絡ありがとうございます。

建設・工事業向けの勤怠管理アプリは、まさに得意分野です。
実際に現場で使われているアプリを開発・運用しておりますので、
現場の実情に合った設計ができます。

よろしければ以下を教えてください：

① 従業員の人数
② 現在の勤怠記録方法（紙・LINEなど）
③ 集計で困っていること（月末の集計作業など）
④ スマホの機種（iPhone / Android）

デモ動画または動作確認URLをお送りすることも可能です。
お気軽にご連絡ください。

AIC開発室
```

```
◯◯様

ご質問ありがとうございます。

ご要件を確認した上で正確なお見積もりをお出ししますが、
目安としては以下のとおりです：

■ シンプルな勤怠管理アプリ（1人〜数名）
  → 80,000円〜、納期7〜10日

■ 複数人対応・クラウド共有あり
  → 120,000円〜、納期14日

■ 勤怠＋日報＋集計レポートなど機能追加
  → 150,000円〜、納期21日

追加機能や細かいカスタマイズについてもご相談を受け付けています。
まずはどんな機能が必要かを教えていただければ、より詳細なお見積もりをお送りします。
```

## テンプレート④：受注確定後の最初のメッセージ

```
◯◯様

ご購入いただきありがとうございます。
早速、制作に向けてヒアリングを進めさせてください。

以下の点を教えていただけますか？

【基本情報】
・会社名または屋号
・業種
・利用人数

【機能について】
・必須の入力項目（例：日付・現場名・開始時間・終了時間・交通費など）
・集計・出力したい内容（月別合計・現場別・CSV出力など）
・管理者（社長・経営者）が確認する画面の必要の有無

【デザイン】
・色やイメージの好み（シンプル / スタイリッシュ / 明るいなど）
・参考にしたいサイト・アプリがあれば

ご回答いただいてから3営業日以内に仕様書（確認用）をお送りします。
よろしくお願いいたします。

AIC開発室
```
